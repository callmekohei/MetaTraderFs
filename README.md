###MetaTraderFs
MetaTraderでFXの取引を個人的に楽にするスクリプトなど

###Install
```
$ git install https://github.com/callmekohei/MetaTraderFs
```

###Illustration
```
(^_^) [MetaTraderFs] $ tree
.
├── README.md
└── code
    ├── BandsCustomed.mq4 // Bollinger Band + EMA10, SMA13, EMA25
    ├── Grid.mq4          // Grid
    └── metatraderFs.fs   // DLL
```

###Tips
windows上でDLLを作る必要があります

target cpu を 32bit にする必要があります
